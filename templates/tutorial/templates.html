{% extends 'tutorial/tutorial_base.html' %} {% block page_title %}② テンプレートを作ろう{% endblock %} {% block page_subtitle %}よく使う支出や収入をテンプレート化することで、記録がもっとラクになります。{% endblock %} {% block content %}

テンプレートを作成しておけば、毎回「名前・金額・収支区分」を入力する必要がありません。
たとえばテンプレートを「水 - 120円（支出）」のようにしておくと、
記録を「水 2」とするだけで2 × 120円で240円の支出となります。

📌 テンプレート操作画面

ホーム画面の「テンプレートの操作」から以下の画面に遷移します。

▲ テンプレート登録と編集・削除の入口

✏️ テンプレートを登録する

「登録をする」ボタンを押すと、以下のようなフォームが表示されます。

▲ テンプレート名・金額・収支区分を入力

🛠 テンプレートを編集・削除する

「操作をする」ボタンを押すと、登録済みテンプレートの一覧が表示されます。編集や削除もここから簡単に行えます。

▲ 編集・削除がワンタップで可能

次へ → 記録を入力する
